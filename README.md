# micro-charts

A tiny (~4 KB) library for visualizing your data. With zero dependencies.

- LineCharts (~2.79 KB)
- BarChart (~2.62 KB)
- LineCharts + BarChart (~3.06 KB)
- PieCharts (~1.44 KB)

\* Minified and gzipped (by [Size Limit](https://github.com/ai/size-limit))
